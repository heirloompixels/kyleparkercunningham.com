{#- works.json — the bridge to the river service. Walks every oeuvre discipline (the year sub-sections are transparent, so each discipline section's .pages already flattens to all its works) and emits one JSON record per work. The river syncs this by slug; the slug is the only contract, so it must be stable and globally unique. Slug = the content path components joined (e.g. "oeuvre/painting/2018/foo"). The bare directory name is NOT unique — two works share the dir name "crystal-tooth-whale" (a 2018 painting and a 2023 print) — and the river keys on slug as a primary key, so a path-based slug is what keeps them distinct. Image = an absolute, base_url-aware URL to the same processed image the site serves, so the river reuses the catalog's images rather than duplicating them. year is emitted as a quoted string to match the river's TEXT column; order is a number. Each json_encode is piped through `safe`: .html templates HTML-escape by default, which would turn json_encode's quotes/slashes into "// and corrupt the JSON. -#} {%- set disciplines = ["painting", "printmaking", "installation", "sculpture", "exhibitions", "other"] -%} {%- set_global works = [] -%} {%- for d in disciplines -%} {%- set sec = get_section(path="oeuvre/" ~ d ~ "/_index.md") -%} {%- set_global works = works | concat(with=sec.pages) -%} {%- endfor -%} [ {%- for p in works %} {%- set image = "" -%} {%- set asset = p.assets | first -%} {%- if asset -%} {%- set resized = resize_image(path=asset, width=960, op="fit_width") -%} {%- set rel = resized.static_path | replace(from="static/", to="") -%} {%- set image = config.base_url ~ "/" ~ rel -%} {%- endif -%} { "slug": {{ p.components | join(sep="/") | json_encode | safe }}, "title": {{ p.title | json_encode | safe }}, "medium": {{ p.extra.medium | default(value="") | json_encode | safe }}, "dim": {{ p.extra.dimensions | default(value="") | json_encode | safe }}, "year": "{{ p.year | default(value="") }}", "image": {{ image | json_encode | safe }}, "order": {{ loop.index }} }{% if not loop.last %},{% endif %} {%- endfor %} ]
